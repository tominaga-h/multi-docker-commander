# v0.1.0 リリースノート

MDC (Multi-Docker-Commander) の初回リリース。
複数リポジトリにまたがる Docker 環境の起動・停止を、1つのコマンドで一括管理・実行できる CLI ツール。

## 主な機能

- **`mdc up` / `mdc down`**: YAML 設定ファイルに定義されたプロジェクト群を一括で起動・停止。プロジェクト間の実行モードとして `parallel` (並列) と `sequential` (直列) を選択可能
- **`mdc list` (`mdc ls`)**: `~/.config/mdc/` 内の設定ファイル一覧を表示 (#7)
- **`mdc procs`**: `mdc up` で起動したバックグラウンドプロセスの状態を一覧表示。設定名による絞り込みにも対応 (#5)
- **`mdc --version` / `mdc -v`**: バージョン情報を表示 (#6)
- **リッチなログ出力**: プロジェクト名プレフィックスと絵文字による状態表示で実行状況を視覚的に把握可能 (#4)
- **バックグラウンド実行**: コマンド単位で `background: true` を指定することで、プロセスをバックグラウンドで起動し PID ファイルで管理 (#5)
- **YAML 設定ファイル**: `~/.config/mdc/` に配置。拡張子 `.yml` の省略に対応。`~` によるホームディレクトリ展開にも対応
