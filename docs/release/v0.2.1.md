# v0.2.1 リリースノート

プロジェクト名を Multi-Docker-Commander（MDC のバクロニム）に変更し、リリース作業を自動化する Cursor Command を追加したメンテナンスリリース。

## 主な機能

- **リリース Cursor Command**: `/release` コマンドによるリリース手順（リリースノート作成・チェンジログ更新・バージョン変更・タグ打ち・GitHub Release 作成）の自動化

## 内部改善

- プロジェクト名を **Multi-Docker-Commander** に変更（README・OVERVIEW・CLI ヘルプ・リリースノート等を一括更新）

## 対応プラットフォーム

- macOS / Linux (PTY 対応)
- Windows (PTY なしのフォールバック)

## 既知の制限

- `mdc edit` コマンドは未実装 (Phase 3 で対応予定)
- `mdc init` コマンドは未実装 (Phase 3 で対応予定)
