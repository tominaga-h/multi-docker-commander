# v0.2.0 リリースノート

Phase 2 の主要機能を実装したリリース。`--dry-run` オプション、`mdc ps` コマンド、`mdc proc attach` コマンド、バックグラウンドプロセスの終了・再起動機能を追加し、実運用に耐えるレベルへ到達。

## 主な機能

- **`--dry-run` オプション**: `mdc up` / `mdc down` 実行時に `--dry-run` を付与すると、実際のコマンドは実行せず実行計画（ディレクトリとコマンドの順序）を表示 (#9)
- **`mdc ps [config-name]`**: 各プロジェクトの Docker コンテナ状態を一覧表示。設定名を省略すると全設定のコンテナを表示 (#16)
- **`mdc proc attach <PID>`**: バックグラウンドプロセスのログ出力をリアルタイムでストリーム。`--tail` / `--no-follow` オプションに対応 (#14)
- **`mdc proc stop <PID>` / `mdc proc restart <PID>`**: バックグラウンドプロセスの停止・再起動をサポート (#13)
